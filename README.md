# The Mau Book

This is the repository with the source code of the Mau documentation, humbly called "The Mau Book".

## Contributions

Please feel free to contribute fixing typos or other mistakes by [submitting PRs](https://github.com/Project-Mau/maubook/pulls) or [opening issues](https://github.com/Project-Mau/maubook/issues).

If you have questions or need clarifications, or simply want to drop an appreciation message, please use the [discussions page](https://github.com/Project-Mau/maubook/discussions).
